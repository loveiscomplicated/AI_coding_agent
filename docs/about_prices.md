  태스크당 토큰 추정

  에이전트별 구조

  ┌─────────────┬────────┬────────────────────┬─────────────────────┬────────────────────────┐
  │  에이전트   │  모델  │     평균 반복      │ Tools 스키마 (토큰) │ System 프롬프트 (토큰) │
  ├─────────────┼────────┼────────────────────┼─────────────────────┼────────────────────────┤
  │ TestWriter  │ Haiku  │ 6회                │ ~1,000 (10개 도구)  │ ~680                   │
  ├─────────────┼────────┼────────────────────┼─────────────────────┼────────────────────────┤
  │ Implementer │ Sonnet │ 10회 (재시도 포함) │ ~1,100 (11개 도구)  │ ~700                   │
  ├─────────────┼────────┼────────────────────┼─────────────────────┼────────────────────────┤
  │ Reviewer    │ Haiku  │ 3회 (max=5)        │ ~800 (8개 도구)     │ ~425                   │
  └─────────────┴────────┴────────────────────┴─────────────────────┴────────────────────────┘

  ---
  태스크당 비용 계산

  캐싱 적용 기준 (1번째 호출: cache write, 이후: cache read):

  TestWriter (Haiku)
  Cache write:  1,680 토큰 × $1.00/MTok  = $0.0017
  Cache read:  8,400 토큰 × $0.08/MTok  = $0.0007
  새 input:    2,400 토큰 × $0.80/MTok  = $0.0019
  Output:      2,100 토큰 × $4.00/MTok  = $0.0084
                                  합계 ≈ $0.014

  Implementer (Sonnet) ← 비용의 85%를 여기서 발생
  Cache write:  1,800 토큰 × $3.75/MTok  = $0.0068
  Cache read:  16,200 토큰 × $0.30/MTok  = $0.0049
  새 input:     4,900 토큰 × $3.00/MTok  = $0.0147
  Output:       7,000 토큰 × $15.00/MTok = $0.105
                                   합계 ≈ $0.131

  Reviewer (Haiku)
  Cache write:  1,625 토큰 × $1.00/MTok  = $0.0016
  Cache read:   3,250 토큰 × $0.08/MTok  = $0.0003
  새 input:       900 토큰 × $0.80/MTok  = $0.0007
  Output:         450 토큰 × $4.00/MTok  = $0.0018
                                  합계 ≈ $0.005

  태스크 1건 합계: ~$0.15

  ---
  월간 비용 시뮬레이션

  태스크 1건 소요 시간: LLM 처리 + Docker 테스트 실행 평균 10~15분

  ┌───────────────────────────┬─────────────┬───────────┬─────────┐
  │         시나리오          │ 태스크/시간 │ 태스크/월 │ 월 비용 │
  ├───────────────────────────┼─────────────┼───────────┼─────────┤
  │ 단순 태스크               │ 6개/h       │ ~4,320    │ ~$430   │
  ├───────────────────────────┼─────────────┼───────────┼─────────┤
  │ 일반 태스크               │ 4개/h       │ ~2,880    │ ~$540   │
  ├───────────────────────────┼─────────────┼───────────┼─────────┤
  │ 복잡 태스크 (재시도 많음) │ 3개/h       │ ~2,160    │ ~$760   │
  └───────────────────────────┴─────────────┴───────────┴─────────┘

  ---
  비용 구조 요약

  Sonnet Output 토큰 → 전체 비용의 ~70%
  Sonnet Input/Cache → ~15%
  Haiku 전체          → ~10%
  기타               → ~5%

  결론: 구현 복잡도(= Implementer 출력 토큰 수) 가 거의 모든 비용을 결정함.

  ---
  추가 절감 레버

  현재 구조에서 더 줄이려면 Implementer만 건드리면 됨:

  1. Implementer를 Sonnet → Haiku로: 태스크당 ~$0.13 → ~$0.02, 총 비용 6~7배 절감 (품질 트레이드오프)
  2. max_iterations 축소 (15→10): 반복이 줄면 output 토큰 비례 감소
  3. 단순 태스크 자동 분류: 간단한 태스크는 Haiku Implementer로 라우팅

  ▎ 참고: 위 가격은 claude-haiku-3-5 / claude-sonnet-3-5 기준 추정치입니다. claude-haiku-4-5 / claude-sonnet-4-6의 실제 가격은 Anthropic 가격 페이지에서 확인하세요. 버전에 따라 ±20~30%
  차이가 날 수 있습니다.
